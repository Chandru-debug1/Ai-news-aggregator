# 🧠 AI News Aggregator

A production-ready AI-powered news aggregator with NLP, ML, and LLM features.

## Features
- Real-time news (Mediastack API)
- Sentiment analysis
- AI summarization (LLM)
- Translation (multi-language)
- Voice narration (gTTS)
- Bias detection
- Fake news classifier

## Tech Stack
Python, Streamlit, NLP, LangChain, OpenAI, Scikit-learn

## Run
pip install -r requirements.txt
python train_model.py
streamlit run app.py
